# Real-Time Chat Application

## Project Description
A full-stack, real-time messaging application designed to provide secure, 1-on-1 private conversations. Built to handle instantaneous data synchronization without requiring page refreshes, the app dynamically routes users into secure private rooms using unique combination IDs.

## Live Demo Link
🔗 [View Live Application Here](https://real-time-chat-app-48582.web.app/) 


## Tech Stack
* **Frontend:** React.js (Vite), JavaScript, CSS
* **Backend & Database:** Firebase Authentication, Cloud Firestore (NoSQL)
* **Hosting:** Firebase Hosting
* **Tools:** npm, Git

## Key Features Built
* **Google Authentication:** Secure, one-click user login and automated profile generation.
* **Dynamic Private Rooms:** Algorithmically generated 1-on-1 chat channels to isolate conversations.
* **Real-Time Sync:** Instant message delivery using Firestore `onSnapshot` listeners.
* **Rich Messaging:** Integrated emoji picker and accurate local message timestamps.
* **Sidebar Previews:** Live preview of the most recent message in the contact list.
